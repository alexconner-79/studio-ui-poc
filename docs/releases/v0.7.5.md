# v0.7.5 -- UI Library & Interface Reskin

**Status:** Complete
**Previous:** [v0.7.0](v0.7.0.md) | **Next:** [v0.8.0](v0.8.0.md)

## Summary

Establishes a proper UI library for the product's own interface -- design tokens, reusable primitives, and consistent styling. Applies a Figma-inspired visual design (purple accent, compact 232px panels, dot-grid canvas, rulers, polished property inspector) across the entire editor shell. Every future release builds UI on top of this foundation.

## Bug Fixes (post-release patch)

- **shadcn Button/Input blue theme** — Root cause: `globals.css` `:root` block redefined `--primary` after `globals-studio.css`, overriding the purple. Fixed by removing `--primary` and `--primary-foreground` from the `globals.css` `:root` and `.dark` blocks.
