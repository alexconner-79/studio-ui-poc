# v0.2.0 -- Editor Polish

**Status:** Complete
**Previous:** [v0.1.0](v0.1.0.md) | **Next:** [v0.3.0](v0.3.0.md)

## Summary

Transformed the basic visual editor into a tool that feels like a real design application. Added layers panel, font management, icon library, image asset management, responsive preview, and keyboard-driven workflow.

## Changelog

### Layers Panel Upgrade

- [x] Rename nodes -- double-click a layer name to edit inline (updates `node.id`)
- [x] Drag-to-reorder in the tree -- reorder layers by dragging within the tree panel
- [x] Visibility toggle -- eye icon per layer to hide nodes in canvas preview (editor-only)
- [x] Lock toggle -- lock icon to prevent accidental edits
- [x] Panel renamed from "Node Tree" to "Layers"

### Font Management

- [x] Google Fonts -- searchable picker, fonts added to project config, compiler emits `<link>` tags
- [x] Custom font import -- upload `.woff2`/`.ttf` files, compiler generates `@font-face`
- [x] Font weight picker (light / regular / medium / semibold / bold)
- [x] Font size, line height, letter spacing, word spacing controls
- [x] Text alignment, decoration, and transform

### Icon Library

- [x] Lucide React bundled (MIT, tree-shakeable)
- [x] Icon node type with `name`, `size`, `color` props
- [x] Searchable icon picker in property panel
- [x] Compiler emits `import { IconName } from "lucide-react"`

### Asset Manager

- [x] Upload API (`POST /api/studio/assets/upload`) -- saves to `public/assets/`
- [x] Asset browser panel with thumbnail grid
- [x] Click to insert or set `src` on selected Image node
- [x] Drag from asset browser to canvas to create Image node

### Responsive Preview

- [x] Breakpoint selector: Mobile (375px), Tablet (768px), Desktop (1280px), Full
- [x] Canvas width constrains to selected breakpoint
- [x] Per-breakpoint prop overrides (e.g., different Grid columns for mobile)

### Keyboard-First Workflow

- [x] `R` rename, `G` group into Stack, `D` duplicate
- [x] `/` command palette -- search actions, node types, screens
- [x] `Cmd+S` save and compile

## Known Issues at Release

- Lucide full `icons` import causes ~2 min initial compile (cached after first run)
- Breakpoints fixed at 3 widths, no custom breakpoint option
