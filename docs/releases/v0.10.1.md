# v0.10.1 — Design System Workspace UX

**Status:** Complete — all scope items shipped.
**Depends on:** v0.10.0 (Design System Authoring)

---

## Overview

The DS workspace and wizard were built as a functional first pass in v0.10.0. This release is a dedicated second pass focused on information architecture, interaction quality, and making the workspace genuinely pleasant to use as a daily tool. The goal is to make it feel as polished as the core canvas editor.

## Known Limitations

- **Component preview (0.10.1.3)**: Static mockup only. Rendering actual npm library components (e.g. real shadcn Button) requires the project to have those packages installed — this is addressed in the Brownfield codebase import path (later release).
- **GitHub export (0.10.1.6)**: PAT is entered inline and never stored server-side. The PAT requires `contents: write` scope on the target repository.
