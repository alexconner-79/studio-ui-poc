# v0.10.7 ‚Äî Advanced Design

**Status:** Complete
**Previous:** [v0.10.6](v0.10.6.md) | **Next:** [v0.10.8](v0.10.8.md)

## Summary

Professional-grade visual design capabilities for Design Mode. Adds multiple fills and strokes per node, a stackable effects system (drop shadow, inner shadow, blur, glass morphism), blend modes, transform controls (rotation, flip, skew), CSS filters (brightness, contrast, saturation, hue), and a full gradient editor. These are the controls designers reach for when creating polished, production-ready screens.

Vector editing (boolean operations, pen tool) is deferred to v1.3.0.

## Design Decisions

**Why CSS `box-shadow` for inside/outside strokes?** CSS `border` only supports center-positioned strokes. Using `box-shadow: inset` for inside strokes and `box-shadow: 0 0 0 Npx` for outside strokes gives identical visual results with full CSS support. The tradeoff is that box-shadow strokes are excluded from the element's layout box (as designed ‚Äî strokes should not shift siblings).

**Why hidden, not disabled, for unsupported framework controls?** Showing a disabled control communicates "this exists but you can't use it now." Hiding it communicates "this doesn't apply to your current context," which is more accurate and less confusing.

**Why a 10-stop maximum on the gradient editor?** More complex gradients should be authored directly in code. 10 stops covers virtually all practical design use cases.

## Known Limitations

- **Boolean operations and pen tool** ‚Äî Addressed in v1.3.0 Advanced Vector Tools.
- **`backdrop-filter` Firefox Android support** ‚Äî Permanent constraint. Browser vendor limitation; the compiler emits the property correctly, the property panel shows a browser compatibility note.
- **Multiple fills/strokes not token-bound** ‚Äî Addressed in v1.0.0 stabilisation pass. Fill/stroke pickers will include a token selector alongside the colour picker.
- **Expo emitter unsupported effects** ‚Äî Addressed in v0.10.7.4 via `FRAMEWORK_CONSTRAINTS` ‚Äî these controls are hidden for Expo projects.
