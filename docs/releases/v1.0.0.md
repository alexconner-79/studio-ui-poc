# v1.0.0 -- Public Release

**Status:** Future
**Previous:** [v0.11.0](v0.11.0.md) | **Next:** [v1.1.0](v1.1.0.md)

## Summary

The launch gate. All v0.x features are stable, documentation is complete, CI/CD is green, and the Vercel deployment is production-ready. This is the version external users see first.

## Launch Criteria

- [ ] All v0.7.0 -- v0.11.0 features implemented and tested
- [ ] No critical bugs in the issue tracker
- [ ] Full test pass by QA on the Vercel deployment
- [ ] User-facing documentation (getting started guide, feature walkthrough)
- [ ] Performance benchmarks: editor loads in under 3 seconds, compile under 5 seconds
- [ ] Supabase RLS policies audited for security
- [ ] OAuth providers (GitHub, Google) tested end-to-end on production domain
- [ ] CLI `create-studio-app` tested with all framework options
- [ ] Monitoring and error tracking in place (e.g., Sentry)

## Scope

No new features. This release is about:

- Stabilisation and bug fixes from v0.7 -- v0.11
- Documentation for end users
- Production environment hardening
- Performance profiling and optimisation
