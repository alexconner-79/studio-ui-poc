# v1.4.0 -- UX Refinement Pass

**Status:** Future
**Previous:** [v1.3.0](v1.3.0.md) | **Next:** --

## Summary

A dedicated second pass over the v0.7.0 Editor UX Overhaul and v0.9.8 Editor Chrome. The first passes established all the core features; this release revisits each sub-section to polish interactions, fix edge cases, improve visual consistency, and incorporate user feedback gathered during real-world usage between v1.0.0 and v1.3.0.

## Scope

### 0.7.1 Revisit -- Project, Screen, and Design System Creation UX

- [ ] Review creation modals for UX consistency and accessibility
- [ ] Improve error handling and validation feedback
- [ ] Add keyboard navigation support (tab order, Enter to submit)
- [ ] Polish rename inline editing (auto-select text, click-away save)

### 0.7.2 Revisit -- Canvas Defaults and Root Container

- [ ] Review default gap/padding values against common design patterns
- [ ] Improve empty-state quick-start options based on usage data
- [ ] Ensure root node styling covers all common properties

### 0.7.3 Revisit -- Interactive Spacing Controls

- [ ] Refine token snapping sensitivity and feedback
- [ ] Improve three-zone overlay hit targets for small elements
- [ ] Polish drag labels (positioning, readability at different zoom levels)
- [ ] Review batch spacing UX (preview before apply, undo support)
- [ ] Test and fix edge cases with nested containers

### 0.7.4 Revisit -- Component Drop Behaviour

- [ ] Improve drop preview ghost sizing and positioning
- [ ] Refine smarter drop targeting visual feedback (parent highlight, tooltip)
- [ ] Review container validation hint accuracy
- [ ] Test drag-and-drop with deeply nested trees

### 0.7.5 Revisit -- Breakpoint Selector

- [ ] Polish full-width breakpoint rendering
- [ ] Improve custom breakpoint input UX (validation, persistence)
- [ ] Review multi-frame layout and spacing
- [ ] Add breakpoint presets for common device sizes

### 0.7.6 Revisit -- Left Menu and Screen Organisation

- [ ] Improve screen search (debounce, highlight matches)
- [ ] Refine screen sort options (add "last updated" when timestamp available)
- [ ] Polish screen groups (collapse state persistence, empty group handling)
- [ ] Refine list/grid toggle (remember preference, transition animation)

### 0.9.8 Revisit -- Onboarding Wizard Second Pass

- [ ] Revisit 5-step wizard based on real user feedback post-launch
- [ ] Improve step transitions and progress indicators
- [ ] Add "reset onboarding" option in Project Settings
- [ ] Review wizard copy and microcopy for clarity

### 0.9.8 Revisit -- Right Panel Deep Pass

- [ ] Comprehensive right panel redesign based on usage patterns observed post-v0.9.8
- [ ] Review contextual section logic for all node types
- [ ] Polish empty-state token quick-reference
- [ ] Keyboard navigation through panel sections
